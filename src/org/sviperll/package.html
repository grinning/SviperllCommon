Package contains basic utilities and extensions for Java SE standard library that can be usefull in everyday programming. The purpose of the library is similar to Google's Guava library.

Packages

  • {@link org.sviperll.cli} Provides tools to parse command line arguments
  • {@link org.sviperll.collection} Provides classes and interfaces for extended, more powerfull collections. Provided collections are fully compatible with standard Java SE collections.
  • {@link org.sviperll.draw} Provides small number of drawing {@see java.awt.Graphics2D} utilities. Wich includes:
    • Versatile text drawing classes
    • Gausian blur filter
  • {@link org.sviperll.io} Provides small number of IO {@see java.io} utilities. Wich includes:
    • Base64 encoding
    • Fixed text line length output (used in Base64 encoding)
    • Truncated to given length streams
  • {@link org.sviperll.mime} Provides multipart/related functionality
  • {@link org.sviperll.property} Provides properties as a library

Classes

  • {@link org.sviperll.Comparables} Allow to abuse exceptions as control flow to write compare/compareTo methods
  • {@link org.sviperll.Comparators} Allow the same exceptions as control-flow for comparators.
  • {@link org.sviperll.CompareException} Exception classes used by {@link org.sviperll.Comparables} and {@link org.sviperll.Comparators}.
  • {@link org.sviperll.Mapper} Interface of mapper, i. e. mathimatical/pure function.
  • {@link org.sviperll.Mappers} Utilities to use with mappers, for example compose function.
  • {@link org.sviperll.Objects} General purpose utilities for objects. Contains helper methods that can be used to write {@link Object#toString}, {@link Object#hashCode} and {@link Object#equals} methods.
  • {@link org.sviperll.Predicate} Interface for functions that can be either true or false.
  • {@link org.sviperll.Predicates} Utilities for predicates, i. e. boolean logic methods.
  • {@link org.sviperll.Reducer} Interface for function that combine two values into one.
  • {@link org.sviperll.Strings} Utilities to use with Strings.
